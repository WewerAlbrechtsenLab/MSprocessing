# MSprocessing

## Installation

MSprocessing runs on Python 3.13 and subsequent compatible versions.

You can install the package directly from GitHub:
```
pip install git+https://github.com/WewerAlbrechtsenLab/MSprocessing.git
```

It is recommended to install InMoose separately using
```
pip install inmoose==0.8.1
```

or using a prebuilt wheel (version 0.8.1, Cpython 3.13, Windows 64bit)
```
pip install "https://raw.githubusercontent.com/a-benedetti/getwheel/main/inmoose-wheels/inmoose-0.8.1-cp313-cp313-win_amd64.whl"
```
